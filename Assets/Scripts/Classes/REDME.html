Данный пакет включает в себя 3 скрипта 1. SoundHolder - менеджер звука. Позволяет задавать набор списков из необходимых для проекта звуков. Для каждого звука имеется возможность настраивать уровень громкости посредством перетягивания ползунка, расположенного справа от него. В оглавлении каждого списка указывается название, с помощью которого в дальнейшем происходит доступ к рандомножу звуку списка. Доступ осуществляется посредством вызова через синглтон функции PlayFromSoundPack(string packName). В том случае, если существует несколько списков с одинаковыминазваниями, выбран будет первый из них. ВАЖНО: Для работоспособности менеджеру ниобходим префаб в виде GameObject'а на котором висит компонент AudioSource. Данный префаб устанавливается в поле "AudioPlayerPrefab" скриптабла SoundHolder через инспектор. 2. DataHolder - абстрактный класс. От данного класса наследуется SoundHolder. Необходим для возможности передачи SoundHolder классу ScriptableInitializator через инспектор. 3. ScriptableInitializator - инициализатор. Необходим для инициализации синглтона SoundHolder. Для корректной работы необходимо установить значение ниже стандартного в Edit > Project settings > Script Execution Order
